Lifestyle-Articles Web Application

Based on the website design of Nelson Sakwa on Behance. Build with the MVC pattern according to the Rails framework, the project has the models, views and controllers of each of the resources User, Article, Vote and Category. The User model can perform any of the CRUD (Create, Read, Update, Delete) operations on the Articles created by the same user and can create and delete a Vote. When an Article is created or updated it can get assigned to a Category, in that way the Article will be grouped among all the articles that share the same category. The most voted article will be displayed in the hero section of the main page. The project has authentication functionality
